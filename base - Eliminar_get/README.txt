Ejemplo de como hacer un registro de datos en php, realizar consultas a una base de datos de mysql, borrar o actualizar registros de la base de datos.

Instalación

1-) Desde el phpMyAdmin, crear la base de datos "registro", con codificación "utf8_unicode_ci".
2-) Desde el phpMyAdmin, importar el fichero registro.sql a la base de datos creada en el paso anterior.
3-) Buscar el proyecto en el localhost y entrar en la carpeta templates del proyecto, esto abrira por defecto el fichero "index.php"
